# spb-housing-market-analysis_datalens
В данном дашборде показаны результаты исследование рынка г. Санкт-Петербурга и области. 
Рассчитаны средние показатели по числу квартир, общей и жилой площади,

Данные по количеству квартир, средней цене за кв. метр и средней длительности продажи показаны в динамике по годам.

Добавлена возможность для фильтрации по наименованию населенного пункта, типу жилья, типу населенного пункта, дате публикации и укрупненно в разрезе года.

Показана взаимосвязь активности публикаций в разрезе дня недели публикации и и месяца публикации.

Детализированы данные по средней площади в разрезе каждого населенного пункта.

Показаны доли распределения объектов жилья по признаку планировки, типу жилья , наличия в радиусе 3 км парков и водоемов.
